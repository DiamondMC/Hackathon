Description

Pigeonhole Sort, is a sorting algorithm in which you create an array of empty "pigeonholes" with size range, which is the largest number minus the smallest plus one. Next it goes through each element individually and puts each one in its pigeonhole, or empty slot. It will repeat the whole thing over and over, adding the non-empty pigeonholes into the original array, until it's finished.

Info

Time Complexity
Average complexity: O(n+range)

Use

Pigeon hole, is a very simple type of sort. It can be used to show the concept of a sorting algorithms, to computer science students. It is also quite efficient, which makes it very good for any beginners
