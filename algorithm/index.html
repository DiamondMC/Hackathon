Algorithm

We assume list is an array of n elements. We further assume that swap function swaps the values of the given array elements.

The algorithim works as follows:
  1. For every element in the list:
  2. Check if the current element is greater than the one to its right
  3. If the left element is greater than swap the elements
  4. Move to the next element
  5. Repeat steps 1-4 until no swaps are made
