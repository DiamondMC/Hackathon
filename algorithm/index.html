Algorithm

We assume list is an array of n elements. and range is the largest number minus the smallest plus 1.

The algorithim works as follows:
  1. Create an array of empty pigeonholes with size range
  2. Check each element in the original array and put it in its pigeonhole
  3. put the non-empty pigeonholes back into the original array
  4. Repeat until it is fully sorted.
